- 克隆仓库
  ```bash
  git clone https://github.com/zqclub/LayerEdge-bot.git
  cd LayerEdge-bot
   ```


## 依赖安装
运行脚本前，需要安装以下依赖：

```bash
npm install

```
钱包设置
示例 =>
```bash
[
    {
      "address": "adrress here",
      "privateKey": "privatekey here"
    },
    {
      "address": "adrress_2 here",
      "privateKey": "privatekey_2 here"
    }
  ]
```
## 运行脚本
在命令行中执行以下命令启动脚本：
```bash
node bot.js
```
